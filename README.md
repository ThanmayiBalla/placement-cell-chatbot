# Placement Cell Chatbot

## Description

A simple rule-based chatbot developed using Python to provide placement-related information to students.

## Features

* Company Information
* Eligibility Criteria
* Placement Training Details
* Internship Information
* Package Details
* Contact Information

## Technologies Used

* Python

## How to Run

1. Open the Python file.
2. Run the program.
3. Ask questions related to placements.
4. Type `bye` to exit.

## Sample Queries

* companies
* eligibility
* training
* internship
* package
* contact
